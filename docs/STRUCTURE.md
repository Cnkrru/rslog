# 文档目录结构

```
docs/
├── zh-CN/                    # 中文文档
│   ├── README.md            # 中文项目概述和快速开始
│   ├── API_GUIDE.md         # 中文API使用指南
│   └── INDEX.md             # 中文文档索引
├── STRUCTURE.md             # 本文档 - 目录结构说明
└── (未来可添加其他语言，如 en-US/, ja-JP/ 等)
```

## 文档管理指南

### 1. 何时更新文档
- 添加新功能时
- 修复重要bug时
- 发布新版本时
- API有重大变化时

### 2. 文档更新流程
```bash
# 使用脚本更新文档
./scripts/update_docs.ps1  # Windows
./scripts/update_docs.sh   # Linux/macOS

# 或手动更新
# 1. 更新代码中的文档注释
# 2. 更新 docs/zh-CN/ 中的文档
# 3. 运行 cargo doc --no-deps 检查警告
# 4. 提交更改
```

### 3. 多语言支持
当前支持：
- 英文：主README.md和代码中的文档注释
- 中文：docs/zh-CN/目录下的文档

未来可扩展支持其他语言。

### 4. 文档同步
确保：
- 中文文档与英文文档内容同步
- 版本号在所有文档中一致
- 示例代码能正常运行

### 5. 发布检查清单
发布新版本前检查：
- [ ] 代码文档注释完整
- [ ] 中文文档已更新
- [ ] 版本号已更新
- [ ] 示例能正常运行
- [ ] 没有文档警告
- [ ] README中的链接有效
